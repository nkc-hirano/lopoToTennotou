
■商品名■

【ロポさん/フルセット】スチームパンク



■フォルダ説明■

　blend　…　Blender 2.83で保存したモデルデータが入ってます。
　fbx　　…　Maya2019を使用し、FBX 2019でエクスポートしたモデルデータが入ってます。
　png　　…　Photoshop 2020にてpng保存したテクスチャデータが入ってます。
　psd　　…　Photoshop 2020で作成したテクスチャのpsd元データが入ってます。
　vrm　　…　Unity 20194.2f1を使用し、UniVRM 0.60で出力したVRMファイルが入ってます。

　ボーン（アーマチュア）
　　┗ UnityのHumanoid形式に対応しています。指ボーンはThumb、Index、Middleの3本です。

　　　　※UnityでRigをHumanoidに設定したとき、Chestボーンだけ認識されないことが多いです。
　　　　　そのときはConfigure画面にて、Chestボーンを手動でChestにドラッグして設定してください。


　VRMファイル
　　┗ 各キャラそれぞれヘッドアクセあり/なしの2種類が入っています。
